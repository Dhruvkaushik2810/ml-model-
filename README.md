# 🩺 MedAlert AI – Disease Prediction System

MedAlert AI is a Machine Learning based disease prediction system developed by **Dhruv Kaushik**.
The system predicts possible diseases based on user-selected symptoms using a **Random Forest Classifier** model.

Built with an interactive **Streamlit** interface for fast and user-friendly predictions.

---

# 🚀 Features

* 🤒 Symptom-based disease prediction
* 🧠 Machine Learning powered predictions
* ⚡ Fast and interactive Streamlit UI
* 📊 Multiple disease support
* 💾 Trained model saved using Joblib
* 🩺 User-friendly medical prediction system

---

# 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Joblib

---

# 🧠 Machine Learning Model

This project uses:

* **Random Forest Classifier**
* Trained on symptom-based medical dataset
* Label Encoding for disease classification

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Dhruvkaushik2810/ml-model-.git
```

Move into the project directory:

```bash
cd ml-model-
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

Start the Streamlit app:

```bash
streamlit run app.py
```

---

# 📸 How It Works

1. User selects symptoms
2. Symptoms are converted into ML input format
3. Random Forest model predicts disease
4. Predicted disease is displayed on screen
